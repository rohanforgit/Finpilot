# UX Rules

Users should never type when AI can infer.

Users should never repeat information twice.

Every important action requires confirmation.

AI should always explain recommendations.

Large financial decisions always require approval.

---

# End State

When users finish any flow they should always feel

"I know exactly where my finances stand."
