# 🎨 FinPilot Design System
Version: 1.0

---

# Design Philosophy

FinPilot should feel like a premium Apple product.

The interface should communicate calmness, trust and simplicity.

Users should never feel overwhelmed.

Every interaction should feel intentional.

Design Inspiration

• Apple
• Linear
• Arc Browser
• Raycast
• Vercel
• Stripe
• Notion

Avoid

• Bootstrap
• Admin Templates
• Traditional Banking Apps
• Heavy Material Design
